# Writing Ideas App

博客选题推荐 Web App，自动从 AI-Informer 日报中评估和推荐高价值写作主题。

## 功能

- 读取 AI-Informer 的日报数据（filtered.json）
- 基于五维评估框架自动打分（受众覆盖度、传播潜力、作者独特性、时效性、系列化潜力）
- Web 界面展示排序后的选题推荐，包含评分详情和建议写作角度

## 使用

```bash
python server.py --port 8081
```

打开 `http://localhost:8081` 查看选题推荐。

## 评分体系

| 维度 | 说明 | 权重 |
|------|------|------|
| 受众覆盖度 | 能触达多少类目标读者 | 1-5 |
| 传播潜力 | 搜索量和社交传播性 | 1-5 |
| 作者独特性 | Tom能否提供独特视角 | 1-5 |
| 时效性 | 发布窗口期 | 1-5 |
| 系列化潜力 | 能否展开为系列内容 | 1-5 |

总分 25 分，20+ 强烈推荐，15-19 值得考虑，<15 优先级低。
